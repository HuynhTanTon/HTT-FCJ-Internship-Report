Mình upload các file frontend lên bucket `url-shortener-frontend-forward`, trong đó:

- `index.html` — giao diện rút gọn link
- `config.js` — cấu hình Function URL của Lambda
- `forest-anime-bg.png` — ảnh nền giao diện
